# Product Overview

Elenchos is an educational platform with a neuro-symbolic approach for teaching exact sciences and programming.

## Core Concept

Combines the flexibility of Large Language Models (LLMs) with the precision of formal mathematical and code verification, using the Socratic method to guide students toward deep understanding.

## Key Features

- **Formal Mathematical Verification**: Rigorous validation using SymPy
- **Secure Code Sandbox**: Isolated execution in Docker containers
- **Socratic Scaffolding**: Gradual help system with 3 levels
- **Personalized RAG**: Integration with teacher content from Notion
- **BKT Engine**: Probabilistic knowledge modeling (Bayesian Knowledge Tracing)
- **ML Analysis**: Archetype clustering and risk prediction

## Target Users

- **Students**: Learning exact sciences and programming
- **Teachers**: Creating classes, problems, and tracking student progress
- **System**: Automated tutoring with AI-powered assistance

## Language

Primary language: Spanish (documentation, UI, content)
